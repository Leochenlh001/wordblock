WordBlock

Break words. Unlock meaning.

WordBlock is a Chrome extension for English vocabulary learning. Select or double-click an English word on any webpage to see definition, etymology hints, morpheme breakdown, and optional AI explanation.

Privacy Policy Support Chrome Web Store

Core Features

  1. Instant word card on text selection
  2. Definition lookup (local-first, online fallback)
  3. Etymology hints from online sources
  4. Prefix/root/suffix breakdown for memory
  5. Optional AI explanation with configurable providers
  6. Lookup history and favorites in popup

Permissions & Data Use

  • storage: saves local cache, history, favorites, and user AI settings.
  • Host permissions: used to run lookup on user-triggered selection and call dictionary/AI endpoints.
  • No ad targeting and no sale of personal data.

For full details, read the Privacy Policy.

How to Use

  1. Install WordBlock from Chrome Web Store.
  2. Select or double-click an English word on any page.
  3. Open extension popup to review history and favorites.
  4. Open options page to enable AI and configure your own API key.
